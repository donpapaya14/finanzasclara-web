## Setting up automatic savings rules
Rocket Money allows you to set up **automatic savings rules**, making it easy to save money without having to think about it. You can create **'Round-Up' rules**, such as saving **$0.50** per coffee purchase, or **'Recurring' transfers**. For example, you can set up a rule to save **$100** every month. According to the [Consumer Financial Protection Bureau auto-savings guide](https://www.consumerfinance.gov), setting up automatic savings rules can help you save **$1,000** in just a few months.
